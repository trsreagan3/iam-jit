about

iam-jit can see only what you submit (chat messages, pasted policies) plus organization context an admin has provided. It cannot see your application code, repos, kubeconfigs, terraform state, the internet, or AWS account contents beyond a read-only discovery role (if your admin enabled it). For code-aware policy authoring, run a local agent on your machine and paste the result via paste mode.
